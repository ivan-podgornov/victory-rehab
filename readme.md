# Victory Rehab

## Развернуть окружение для разработки

```
git clone https://github.com/ivan-podgornov/victory-rehab
cd ./victory-rehab
npm install
```

## Команды

* **npm start** - запустить проект в режиме разработки
* **npm run production** - запустить проект в боевом окружении
